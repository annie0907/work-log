📌 新增資料夾的方法（網頁版）

在 2025/ 頁面右上角，點選綠色的 Add file → Create new file。

在檔名輸入框輸入：

10/README.md


10/ 代表建立一個資料夾名稱為 10。

README.md 是放在裡面的檔案（必須有一個檔案，GitHub 才會建立資料夾）。

在編輯區隨便輸入一行字，例如：

# 2025/10 月份紀錄


下方 Commit message 輸入：

Create 2025/10 folder


點擊 Commit new file。

這樣就會在 2025/ 底下新增一個 10 資料夾。
